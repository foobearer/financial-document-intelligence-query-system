# Sample Documents

Download real SEC filings to test FinDocIQ.

## Where to get free financial PDFs

**SEC EDGAR (official source)**
https://www.sec.gov/cgi-bin/browse-edgar

Search for any public company, click "10-K" or "10-Q",
then download the filing as a PDF.

**Suggested test companies:**
- Apple Inc (AAPL) — CIK: 0000320193
- Tesla Inc (TSLA) — CIK: 0001318605
- Microsoft Corp (MSFT) — CIK: 0000789019
- Amazon.com Inc (AMZN) — CIK: 0001018724

## Earnings Transcripts

Seeking Alpha and The Motley Fool publish free earnings call transcripts.
Search "[Company Name] Q1 2024 earnings call transcript PDF".
